项目名称：南方学生会管理系统

开发周期：：2018.11 – 2018.03

项目类型：个人项目

开发语言：Java、MySQL

开发工具：IDEA、HBuilder、Adobe Photoshop CC

后台框架：Spring、SpringMVC、MyBatis

前端框架：Vue-Router、iView、ECharts、axios

其他：腾讯云服务器、腾讯云短信 API、Log4j、Shiro

项目介绍：

     二学年项目，使用 Maven、GitHub、Vue、MySql、SSM、Echarts、log4j 等开发，该项目实现前后台分离，后端 API 接口基本实现，下一步做出手机 app 端，利用 Spring AOP 实现权限的管理，使用 Echarts 进行报表，使用 github 同步项目，正在对该项目 APP 端下一步进行分析。

项目心得：

     个人项目设计出来后，请教同学和老师，都有各自观点，需要分析很多的的功能，从客户角度考虑，同时在分析过程中有很多的收获，比如开会考勤，我们能不能使用手机扫描本次开会的二维码直接记录考勤，考虑到有些学校是不能携带手机的，该如何去设计，让系统更加的完善等方面。只有通过实践才能有出收获，而一个项目的创建并不是简简单单的进行编程而已。在动手之前，一定要分析好相关业务逻辑，整理好功能需求，明确目标。做项目之前理清思路可以帮助我们快速编程，防止编程中思维混乱，或者发现某部分功能不符合需求，删掉重来的情况。
